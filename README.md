# 🍔 Fast Bite - Restaurant Management System (C++ OOP)

## 📌 Project Overview
Fast Bite is an Object-Oriented Programming (OOP) based Restaurant Management System developed in C++.  
The project simulates a real-world fast-food ordering system where users can explore menu items, place orders, apply discounts, and generate detailed bills.

This project demonstrates core OOP concepts including inheritance, encapsulation, friend functions, composition/aggregation, pointers, and error handling.

---


## ✨ Features

- 🍟 Display restaurant menu with item names and prices
- 🛒 Select food items and quantities
- 🧾 Generate itemized receipts
- 💰 Automatic bill calculation
- 🏷️ Apply discounts
- 📊 Calculate subtotal, tax, and total amount
- ✅ Input validation and error handling
- 👨‍🍳 User-friendly console interface

---

## 🧠 OOP Concepts Used

- Classes & Objects
- Encapsulation
- Inheritance
- Friend Functions
- Composition / Aggregation
- Pointers
- Error Handling

---

## ⚙️ Working Algorithm

1. User selects the type of order
2. System displays available menu items
3. User selects items and enters quantities
4. Input validation checks incorrect entries
5. Selected items are added to the order
6. Subtotal is calculated
7. Discount is applied if available
8. Tax is added
9. Final receipt is generated
